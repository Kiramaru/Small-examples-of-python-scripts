# Small-examples-of-python-scripts

FindingLargestCommonDivisorInTheRange
На вход 2 целых числа: начало и конец диапазона, диапазон заполняется с шагом 1 и считается НОД

FindingTwoMinimumNumbersAndTheirSum
На вход диапазон через пробел, данные могут быть любыми, если целых чисел меньше 2, то программа не отрабатывает
Происходит поиск 2 целых минимальных чисел, вычисляется их сумма и выводятся сами числа и сумма

RemovingConsecutiveDuplicatesFromAnArray
Функция, которая принимает на вход массив и убирает дубликаты, идущие подряд

AutotestsForTheCompress_NumbersFunction
Автотесты для функции, которая убирает дубликаты
